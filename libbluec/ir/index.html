Module ir

Expand description

The ir module is responsible for lowering the C AST into the “BlueTac” intermediate representation (IR).

BlueTac is a custom three-address code (TAC) IR for the BlueC compiler. In the future, we’ll add a Static Single Assignment (SSA) constraint.
